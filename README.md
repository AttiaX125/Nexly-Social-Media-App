## 🌟 Overview
**Nexly** is a modern, responsive social app built with **React**, **Bootstrap**, and **Font Awesome**. It allows users to register, create posts, interact with others, and enjoy a smooth, mobile-friendly experience.  

This project demonstrates clean React architecture, form validation, and professional deployment practices.

---

## 🎯 Features
- 🔹 User registration & login
- 🔹 Create, edit, and delete posts
- 🔹 Like and comment on posts
- 🔹 Responsive mobile & desktop design
- 🔹 React Router for smooth navigation
- 🔹 Form validation with **Joi**
- 🔹 Fully deployed on **Vercel**

---

## 🛠 Tech Stack
| Layer        | Technology                  |
|--------------|----------------------------|
| Frontend     | React, Bootstrap, Font Awesome |
| Routing      | React Router               |
